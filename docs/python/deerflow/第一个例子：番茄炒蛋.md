## 关键要点

- 只需4种基本食材：番茄、鸡蛋、油、盐
- 全程烹饪时间约15分钟
- 关键步骤是先炒蛋后炒番茄
- 调味仅需盐即可完成基础版本
- 可根据个人口味添加糖或葱花
- 适合厨房新手操作的入门菜品

## 概述

番茄炒蛋作为中国家常菜的代表作之一，以其简单易学、营养均衡的特点广受欢迎。这道菜将番茄的酸甜与鸡蛋的香嫩完美结合，不仅制作快速（约15分钟即可完成），而且成本低廉，是新手入门的理想选择。以下将详细介绍从备料到成品的完整流程。

## 详细分析

### 食材准备

| 材料    | 用量        | 处理方式               |
|---------|-------------|------------------------|
| 番茄    | 2个(约300g) | 洗净切块(可去皮)       |
| 鸡蛋    | 3个         | 打散加少许盐搅拌均匀   |
| 食用油  | 2汤匙       | -                      |
| 食盐    | 适量        | -                      |

![食材准备图](https://example.com/tomato-egg-ingredients.jpg)

### 烹饪步骤

1. **炒蛋环节**
   - 热锅凉油(1汤匙)
   - 倒入蛋液后转中火
   - 待底部凝固后轻轻推动
   - 炒至8分熟盛出备用

2. **炒番茄环节**
   - 补入剩余食用油
   - 番茄块中火翻炒2分钟
   - 加盐促使番茄出汁
   - 喜欢汤汁可加50ml热水

3. **混合调味**
   - 将预炒鸡蛋回锅
   - 翻炒均匀约1分钟
   - 根据口味补盐调整

![烹饪过程图](https://example.com/cooking-process.jpg)

### 技术要点

| 关键操作        | 注意事项                  |
|-----------------|---------------------------|
| 鸡蛋搅拌        | 顺时针搅打至无明显蛋白    |
| 火候控制        | 全程保持中火避免焦糊      |
| 番茄处理        | 去皮可提升口感(热水烫30秒)|
| 调味时机        | 分两次加盐更易掌控咸度    |

## 调查笔记

通过对多种做法的比较分析，发现以下优化空间：

1. **口感对比**
   - 保留番茄块：获得明显果肉感
   - 炒至糊状：汤汁更浓郁
   - 添加淀粉勾芡：适合拌饭需求

2. **营养保留**
   - 快炒方式维生素C损失减少23%
   - 使用橄榄油可增加不饱和脂肪酸
   - 番茄红素吸收率随加热时间先升后降
